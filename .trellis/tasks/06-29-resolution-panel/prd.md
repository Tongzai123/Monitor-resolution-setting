# 分辨率主面板接入

## 目标

在现有托盘主面板显示每台显示器当前分辨率/刷新率，并提供模式选择、切换和确认回滚 UI。

## 需求

- 分辨率入口必须附着在现有显示器卡片中。
- 不改变用户调整亮度的主要流程。
- 当前模式显示为紧凑文本，例如 `3840x2160 · 60Hz`。
- 分辨率入口采用类似 Windows 11 快捷设置的分段展开组件：左侧显示当前模式，右侧小箭头展开模式列表。
- 点击小箭头后展开模式列表，按收藏/常用/全部模式分组。
- 当前模式有明确选中态。
- 切换后显示确认条，包含“保留更改”“还原”和倒计时。
- 面板关闭后倒计时由主进程继续，重新打开面板时应显示最新状态。

## 验收标准

- [ ] 每台支持 Win32 显示配置的显示器显示当前模式。
- [ ] 分辨率行不遮挡或挤压亮度滑块。
- [ ] 分辨率组件右侧有明确小箭头，点击后展开模式列表。
- [ ] 模式列表过长时可滚动。
- [ ] 切换成功后确认条倒计时实时更新。
- [ ] 用户可从 UI 主动还原。
- [ ] 切换失败时有清晰错误提示。
