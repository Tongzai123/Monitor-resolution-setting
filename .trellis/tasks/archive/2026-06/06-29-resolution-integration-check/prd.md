# 分辨率完整首版集成验收

## 目标

对完整首版做端到端验收，确认底层、主面板、设置收藏、右键菜单和快捷键共同工作，并且没有破坏 Twinkle Tray 原有核心功能。

## 需求

- 完整跑通五阶段验收标准。
- 验证真实显示器、多显示器和功能关闭场景。
- 验证失败路径：切换失败、回滚失败、快捷键注册失败。
- 验证现有亮度控制、设置保存、托盘显示不回归。
- 整理残余风险和后续建议。
- 打包生成可直接双击运行的 Windows `.exe` 最终产物，并记录输出路径。

## 验收标准

- [ ] 父任务所有验收标准通过或有明确残余风险说明。
- [ ] 构建通过。
- [ ] 主要手工验证记录完整。
- [ ] 未引入密钥、临时调试输出或无关重构。
- [ ] 功能可通过设置总开关关闭。
- [ ] 已生成可直接双击运行的 `.exe` 最终产物，且验收记录包含准确输出路径。
