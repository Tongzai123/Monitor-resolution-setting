# 分辨率完整首版集成验收设计

## 验收矩阵

覆盖单屏、多屏、收藏为空、收藏存在、功能关闭、低刷新率隐藏、面板关闭后回滚、右键菜单切换、快捷键切换。

## 回归范围

重点检查亮度滑块、HDR SDR 滑块、DDC/CI 扩展功能、设置保存、托盘菜单、全局快捷键。

## 验收记录

集成验收必须新增或更新 `verification.md`，记录：

- 验证日期、系统环境、显示器数量和关键显示器信息。
- 构建命令和结果。
- 单屏、多屏、收藏为空、收藏存在、功能关闭、失败路径的手工验证结果。
- 残余风险和后续建议。
- 最终 `.exe` 产物路径。

## 打包产物

使用现有 Electron Builder 流程生成 Windows `.exe`。优先顺序：

1. 运行 `npm run parcel-build` 生成前端资源。
2. 运行 `npm run electron-build` 生成 Windows 安装包或可执行产物。
3. 检查输出目录中的 `.exe` 文件存在，并把完整路径写入 `verification.md`。
