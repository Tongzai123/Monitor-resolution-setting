# 分辨率设置与收藏

## 目标

提供分辨率功能开关、模式筛选选项和按显示器保存的收藏模式。

## 需求

- 设置页增加分辨率区域。
- 支持总开关：在主面板显示分辨率控制。
- 支持显示/隐藏刷新率文本。
- 支持默认隐藏低于 60Hz 的模式。
- 支持显示所有 Windows 报告模式。
- 支持只显示收藏模式。
- 支持配置自动还原倒计时。
- 支持按显示器收藏/取消收藏模式。
- 收藏重启后保持，多显示器互不干扰。
- 预留场景预设数据结构。

## 验收标准

- [ ] 关闭总开关后主面板不显示分辨率入口。
- [ ] 设置变更通过现有设置保存机制持久化。
- [ ] 收藏模式显示在模式列表顶部。
- [ ] 只显示收藏模式时，非收藏模式不出现在主列表。
- [ ] 重启应用后收藏仍可用。
