# Twinkle Tray 分辨率增强版 v1.17.2-2

## 更新内容

- 优化分辨率切换确认体验：切换分辨率后，“保留更改/还原”确认条显示期间，悬浮面板会保持可见并置顶。
- 修复确认倒计时期间面板误隐藏问题：点击桌面、其他窗口或任务栏时，确认面板不再消失，用户无需重新点击托盘找回确认按钮。
- 修复主动隐藏路径绕过问题：确认倒计时期间，按 `Escape` 或触发面板失焦、隐藏动画等路径时，面板也会继续保持显示。
- 保留原有安全回滚机制：用户点击“保留更改”“还原”或倒计时结束后，面板会恢复普通行为，后续点击外部可正常隐藏。
- 优化便携版压缩包结构：便携版 `.zip` 解压后会先得到一个外层文件夹，避免程序文件直接散落到当前目录。
- 更新 GitHub Release 构建流程：通过 `Run workflow` 构建出的便携版 `.zip` 也会包含外层文件夹，和本地构建结果保持一致。
- 更新版本号为 `v1.17.2-2`，便于和 `v1.17.2-1` 发布版本区分。

## 下载说明

- 安装版：`Twinkle.Tray.Resolution.Enhanced.v1.17.2-2.exe`
- 便携版：`Twinkle.Tray.Resolution.Enhanced.v1.17.2-2-portable.zip`

## 升级建议

- 如果你使用安装版，直接下载并运行 `.exe` 安装包即可。
- 如果你使用便携版，建议完整解压新版 `.zip`。新版压缩包会自动包含外层文件夹，解压后可直接进入文件夹运行 `Twinkle Tray.exe`。
- 如果你经常切换分辨率，建议升级到此版本，以避免确认倒计时期间面板被误隐藏导致自动还原。
