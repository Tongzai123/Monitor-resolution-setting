# Twinkle Tray 分辨率增强版 v1.17.2-1

## 更新内容

- 优化分辨率列表显示逻辑：在不显示刷新率时，同一分辨率会自动去重，优先保留更高刷新率或当前正在使用的模式，列表更清爽。
- 优化常用分辨率切换体验：从右键菜单或快捷键切换已收藏的常用分辨率时，不再弹出确认/回滚提示，操作更直接。
- 保留普通分辨率切换的安全机制：非收藏模式仍会进入确认/自动回滚流程，避免误切换后黑屏或显示异常。
- 优化分辨率下拉列表体验：再次展开已扫描过的显示器时，会优先显示缓存结果，同时后台刷新最新模式，减少“加载中”闪烁。
- 优化深色主题下的分辨率列表滚动条样式：滚动块改为灰色，滚动条背景跟随主题，不再出现突兀的白色背景。
- 优化便携版数据目录识别：便携版会在程序目录下使用独立 `config` 数据目录，减少与安装版配置互相影响。
- 优化安装包行为：安装完成后自动运行程序；重装时会重新创建桌面快捷方式。
- 更新版本号为 `v1.17.2-1`，便于和上一个 `v1.17.2` 发布版本区分。

## 下载说明

- 安装版：`Twinkle.Tray.Resolution.Enhanced.v1.17.2-1.exe`
- 便携版：`Twinkle.Tray.Resolution.Enhanced.v1.17.2-1-portable.zip`

## 升级建议

- 如果你使用安装版，直接下载并运行 `.exe` 安装包即可。
- 如果你使用便携版，建议完整解压新版 `.zip` 后运行，配置会保存在程序目录下的 `config` 文件夹中。
- 如果你依赖常用分辨率快捷键或右键菜单，建议升级到此版本，以获得更顺滑的收藏分辨率切换体验。
